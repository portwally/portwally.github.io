Privacy Policy

CyanHero

CyanHero does not collect, transmit, or store any of your personal data. This app is designed with complete privacy protection. All image processing happens entirely on your local computer, and no information ever leaves your device.

Data Collection

CyanHero collects ZERO data. Specifically:

  • No personal information is collected
  • No user analytics or telemetry
  • No tracking of app usage
  • No network connections or internet communication
  • No crash reports or diagnostic data sent to developers
  • No advertising identifiers or cookies

Image Processing

All image processing operations are performed entirely on your local Mac:

  • Local Processing: Images are processed using your Mac's CPU and GPU only
  • No Cloud Services: Images are never uploaded to any server or cloud service
  • Your Files Stay Yours: Only you have access to your images and negatives
  • Complete Control: You choose where files are saved on your computer

File Access

CyanHero only accesses files that you explicitly select:

  • The app can only read images you choose to import
  • The app can only write files to locations you authorize
  • All file access is sandboxed according to macOS security standards
  • No background file scanning or indexing occurs

Network Activity

CyanHero has ZERO network activity:

  • The app does not connect to the internet
  • No data is transmitted to any external servers
  • The app does not check for updates automatically
  • No third-party services or APIs are used

Curve Files and Presets

Tone curve files (.acv) are stored and processed locally:

  • Bundled preset curves are included in the app package
  • Custom curves you load or save remain on your computer only
  • No curve data is shared or synchronized

Print Settings

Print configuration and page layout settings:

  • Print settings are saved locally using macOS standard preferences
  • These settings are never transmitted outside your device
  • You can clear these settings at any time through macOS settings

Third-Party Services

CyanHero uses NO third-party services:

  • No analytics frameworks (e.g., Google Analytics, Mixpanel)
  • No crash reporting services (e.g., Crashlytics, Sentry)
  • No advertising networks or SDKs
  • No social media integrations
  • No cloud storage providers

App Store Distribution

If you purchase CyanHero from the Mac App Store:

  • Apple processes your payment and transaction data according to their privacy policy
  • The developer does not receive any personal payment information
  • CyanHero itself still collects zero data on your device

Open Source Transparency

CyanHero's source code is available on GitHub:

  • You can review the complete source code to verify privacy claims
  • The code is open for security audits and community review
  • No hidden code or obfuscated functionality

Changes to This Policy

Any changes to this privacy policy will be:

  • Posted on this page with an updated date
  • Announced on the GitHub repository
  • Will never compromise the core principle: zero data collection

Contact

If you have questions about this privacy policy, you can:

  • Open an issue on the GitHub repository
  • Contact the developer through the support channels listed on the CyanHero website

Privacy Commitment

CyanHero is built on a foundation of complete user privacy. Your images, your work, and your creative process are entirely your own. This app exists solely as a tool to help you create beautiful alternative process prints, and it does so without compromising your privacy in any way.
