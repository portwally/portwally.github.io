Why it's different

Not a sledgehammer. A scalpel.

Generic cleaners don't know the difference between your chat history and a 40 GB cache blob. Clean AI does — because every path was audited by hand.

↺

Trash, never permanent

Every selected item is moved via FileManager.trashItem. If you change your mind, restore from the Trash like any other file.

◉

Audited paths

Every scan location was verified on a real machine. We know which files are regenerable caches and which hold your settings — and label them accordingly.

○

No account. No telemetry.

Runs entirely on your Mac. Nothing leaves your machine.

⇅

Per-app views

List and chart views for each app. Sort by size. Select what to remove individually.

⟳

Smart duplicates

When you have two versions of Claude Code installed, the older one is flagged as safe to delete automatically.

✻

Safety model

Three colors. No ambiguity.

Every scanned path is classified before you see it. The same colors appear throughout the app — green, orange, red — and the UI refuses to let you select the red ones.

Safe
Regenerated automatically. HTTP caches, compiled workers, log files, temp directories, downloaded extension packages. Delete without concern.
Selectable by default
Caution
Removable, but with consequences. Model weights require re-download (gigabytes). Extensions need reinstalling. Workspace storage loses chat and search history.
Selectable, clearly warned
Do not delete
Settings, credentials, configuration. OAuth tokens, user preferences, MCP configs, plugin settings. Shown for visibility but the UI will not let you select them.
Lock icon, non-selectable
Flow

Scan. Review. Trash.

No configuration, no profiles, no preset cleanup recipes. Just three steps you can stop at any time.

Pick an app

The sidebar shows every supported app with its real icon. Tap one — Clean AI scans it in the background and shows every discovered item sorted by size.

Review & select

See each path with its category, safety level, and size. Flip between a grouped list and a horizontal bar chart. Select everything safe with one click, or pick individually.

Move to Trash

A final confirmation shows exactly how many items and how many megabytes. Clean AI moves them to the Trash and re-scans so you can see what's left.
