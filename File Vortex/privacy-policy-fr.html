Politique de Confidentialité

File Vortex

← Retour à File Vortex

File Vortex a été conçu avec la confidentialité comme principe fondamental. Tout le traitement des fichiers et l'analyse IA se déroulent entièrement sur votre appareil. Aucune donnée n'est collectée, transmise ou partagée avec quiconque.

Collecte de Données

File Vortex ne collecte, ne stocke ni ne transmet aucune donnée personnelle. L'application fonctionne entièrement hors ligne et localement sur votre Mac.

Plus précisément, File Vortex ne collecte pas :

  • Informations personnelles (nom, e-mail, adresse, etc.)
  • Analyses d'utilisation ou télémétrie
  • Contenu ou métadonnées des fichiers
  • Rapports de plantage ou diagnostics
  • Données de localisation
  • Identifiants d'appareil

Traitement sur l'Appareil

Toutes les analyses de fichiers, y compris la reconnaissance d'image par IA, l'extraction de texte et la génération de mots-clés, sont effectuées entièrement sur votre Mac en utilisant les frameworks Vision et NaturalLanguage intégrés d'Apple. Aucun fichier, image ou donnée extraite n'est jamais envoyé à des serveurs externes.

Stockage Local des Données

File Vortex stocke les données suivantes localement sur votre Mac :

  • Base de données d'index de fichiers : Une base de données SwiftData locale contenant les métadonnées de vos fichiers organisés (noms de fichiers, tags, descriptions, miniatures). Ces données restent sur votre appareil.
  • Préférences utilisateur : Vos paramètres tels que le chemin du dossier de stockage et la préférence de format de date, stockés dans les préférences locales de l'application.
  • Signets à portée de sécurité : Données de signets macOS qui permettent à l'application d'accéder au dossier de stockage choisi entre les lancements de l'application.

Toutes les données stockées localement peuvent être supprimées en désinstallant l'application.

Accès aux Fichiers

File Vortex accède aux fichiers uniquement lorsque vous les glissez-déposez explicitement sur la fenêtre de l'application, ou lorsque vous recherchez dans votre bibliothèque indexée. L'application accède au dossier de stockage choisi (NAS, disque externe ou dossier local) pour copier et organiser les fichiers. Cet accès est accordé par vous via le sélecteur de fichiers macOS et est protégé par le sandboxing macOS.

Accès Réseau

File Vortex nécessite un accès réseau uniquement pour communiquer avec votre stockage réseau local (NAS) via le protocole SMB. L'application ne se connecte à aucun serveur externe, API ou service cloud.

Services Tiers

File Vortex n'utilise aucun service tiers, SDK, outil d'analyse ou framework publicitaire.

Confidentialité des Enfants

File Vortex ne collecte aucune donnée d'aucun utilisateur, y compris les enfants. L'application est sûre pour une utilisation par des personnes de tous âges.

Modifications de cette Politique

Si cette politique de confidentialité est mise à jour, les modifications seront publiées sur cette page avec une date d'effet mise à jour. Étant donné que File Vortex ne collecte aucune donnée, des changements significatifs sont peu probables.
