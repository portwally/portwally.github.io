Datenschutzrichtlinie

File Vortex

← Zurück zu File Vortex

File Vortex wurde mit Datenschutz als Kernprinzip entwickelt. Alle Dateiverarbeitung und KI-Analyse erfolgt vollständig auf Ihrem Gerät. Es werden keine Daten gesammelt, übertragen oder mit Dritten geteilt.

Datenerfassung

File Vortex erfasst, speichert oder überträgt keine personenbezogenen Daten. Die App arbeitet vollständig offline und lokal auf Ihrem Mac.

File Vortex erfasst insbesondere nicht:

  • Persönliche Informationen (Name, E-Mail, Adresse usw.)
  • Nutzungsanalysen oder Telemetrie
  • Dateiinhalte oder Metadaten
  • Absturzberichte oder Diagnosen
  • Standortdaten
  • Gerätekennungen

Verarbeitung auf dem Gerät

Alle Dateianalysen, einschließlich KI-gestützter Bilderkennung, Textextraktion und Schlüsselwortgenerierung, werden vollständig auf Ihrem Mac mit Apples integrierten Vision- und NaturalLanguage-Frameworks durchgeführt. Keine Dateien, Bilder oder extrahierten Daten werden jemals an externe Server gesendet.

Lokale Datenspeicherung

File Vortex speichert folgende Daten lokal auf Ihrem Mac:

  • Dateiindex-Datenbank: Eine lokale SwiftData-Datenbank mit Metadaten über Ihre organisierten Dateien (Dateinamen, Tags, Beschreibungen, Miniaturansichten). Diese Daten bleiben auf Ihrem Gerät.
  • Benutzereinstellungen: Ihre Einstellungen wie der Speicherordnerpfad und die Datumsformateinstellung, gespeichert in den lokalen Einstellungen der App.
  • Sicherheitsbereichs-Lesezeichen: macOS-Lesezeichendaten, die der App den Zugriff auf Ihren gewählten Speicherordner über App-Neustarts hinweg ermöglichen.

Alle lokal gespeicherten Daten können durch Löschen der App entfernt werden.

Dateizugriff

File Vortex greift nur auf Dateien zu, wenn Sie diese explizit per Drag & Drop auf das App-Fenster ziehen oder wenn Sie Ihre indizierte Bibliothek durchsuchen. Die App greift auf Ihren gewählten Speicherordner (NAS, externes Laufwerk oder lokaler Ordner) zu, um Dateien zu kopieren und zu organisieren. Dieser Zugriff wird von Ihnen über die macOS-Dateiauswahl gewährt und ist durch macOS-Sandboxing geschützt.

Netzwerkzugriff

File Vortex benötigt Netzwerkzugriff ausschließlich für die Kommunikation mit Ihrem lokalen Netzwerkspeicher (NAS) über das SMB-Protokoll. Die App verbindet sich nicht mit externen Servern, APIs oder Cloud-Diensten.

Drittanbieterdienste

File Vortex verwendet keine Drittanbieterdienste, SDKs, Analysetools oder Werbe-Frameworks.

Datenschutz für Kinder

File Vortex erfasst keine Daten von Nutzern, einschließlich Kindern. Die App ist für Personen jeden Alters sicher nutzbar.

Änderungen dieser Richtlinie

Wenn diese Datenschutzrichtlinie aktualisiert wird, werden die Änderungen auf dieser Seite mit einem aktualisierten Gültigkeitsdatum veröffentlicht. Da File Vortex keine Daten erfasst, sind wesentliche Änderungen unwahrscheinlich.
