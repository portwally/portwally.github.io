Política de Privacidade

File Vortex

← Voltar ao File Vortex

O File Vortex foi projetado com a privacidade como princípio fundamental. Todo o processamento de arquivos e análise de IA acontece inteiramente no seu dispositivo. Nenhum dado é coletado, transmitido ou compartilhado com terceiros.

Coleta de Dados

O File Vortex não coleta, armazena ou transmite quaisquer dados pessoais. O aplicativo opera inteiramente offline e localmente no seu Mac.

Especificamente, o File Vortex não coleta:

  • Informações pessoais (nome, e-mail, endereço, etc.)
  • Análises de uso ou telemetria
  • Conteúdo ou metadados de arquivos
  • Relatórios de falhas ou diagnósticos
  • Dados de localização
  • Identificadores de dispositivo

Processamento no Dispositivo

Toda a análise de arquivos, incluindo reconhecimento de imagem por IA, extração de texto e geração de palavras-chave, é realizada inteiramente no seu Mac usando os frameworks Vision e NaturalLanguage integrados da Apple. Nenhum arquivo, imagem ou dado extraído é enviado para servidores externos.

Armazenamento Local de Dados

O File Vortex armazena os seguintes dados localmente no seu Mac:

  • Banco de dados de índice de arquivos: Um banco de dados SwiftData local contendo metadados sobre seus arquivos organizados (nomes de arquivos, tags, descrições, miniaturas). Esses dados permanecem no seu dispositivo.
  • Preferências do usuário: Suas configurações, como o caminho da pasta de armazenamento e preferência de formato de data, armazenadas nas preferências locais do aplicativo.
  • Marcadores com escopo de segurança: Dados de marcadores do macOS que permitem que o aplicativo acesse a pasta de armazenamento escolhida entre reinicializações do aplicativo.

Todos os dados armazenados localmente podem ser removidos excluindo o aplicativo.

Acesso a Arquivos

O File Vortex acessa arquivos apenas quando você os arrasta e solta explicitamente na janela do aplicativo, ou quando você pesquisa sua biblioteca indexada. O aplicativo acessa a pasta de armazenamento escolhida (NAS, unidade externa ou pasta local) para copiar e organizar arquivos. Esse acesso é concedido por você através do seletor de arquivos do macOS e é protegido pelo sandboxing do macOS.

Acesso à Rede

O File Vortex requer acesso à rede exclusivamente para comunicação com seu armazenamento de rede local (NAS) via protocolo SMB. O aplicativo não se conecta a servidores externos, APIs ou serviços de nuvem.

Serviços de Terceiros

O File Vortex não usa serviços de terceiros, SDKs, ferramentas de análise ou frameworks de publicidade.

Privacidade de Crianças

O File Vortex não coleta dados de nenhum usuário, incluindo crianças. O aplicativo é seguro para uso por pessoas de todas as idades.

Alterações nesta Política

Se esta política de privacidade for atualizada, as alterações serão publicadas nesta página com uma data de vigência atualizada. Como o File Vortex não coleta nenhum dado, mudanças significativas são improváveis.
