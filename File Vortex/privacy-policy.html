Privacy Policy

File Vortex is designed with privacy at its core. All file processing and AI analysis happens entirely on your device. No data is collected, transmitted, or shared with anyone.

Data Collection

File Vortex does not collect, store, or transmit any personal data. The app operates entirely offline and locally on your Mac.

Specifically, File Vortex does not collect:

  • Personal information (name, email, address, etc.)
  • Usage analytics or telemetry
  • File contents or metadata
  • Crash reports or diagnostics
  • Location data
  • Device identifiers

On-Device Processing

All file analysis, including AI-powered image recognition, text extraction, and keyword generation, is performed entirely on your Mac using Apple's built-in Vision and NaturalLanguage frameworks. No files, images, or extracted data are ever sent to external servers.

Local Data Storage

File Vortex stores the following data locally on your Mac:

  • File index database: A local SwiftData database containing metadata about your organized files (filenames, tags, descriptions, thumbnails). This data stays on your device.
  • User preferences: Your settings such as the storage folder path and date format preference, stored in the app's local preferences.
  • Security-scoped bookmarks: macOS bookmark data that allows the app to access your chosen storage folder across app launches.

All locally stored data can be removed by deleting the app.

File Access

File Vortex accesses files only when you explicitly drag and drop them onto the app window, or when you search your indexed library. The app accesses your chosen storage folder (NAS, external drive, or local folder) to copy and organize files. This access is granted by you through the macOS file picker and is protected by macOS sandboxing.

Network Access

File Vortex requires network access solely to communicate with your local network storage (NAS) via the SMB protocol. The app does not connect to any external servers, APIs, or cloud services.

Third-Party Services

File Vortex does not use any third-party services, SDKs, analytics tools, or advertising frameworks.

Children's Privacy

File Vortex does not collect any data from any user, including children. The app is safe for use by people of all ages.

Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page with an updated effective date. Since File Vortex does not collect any data, significant changes are unlikely.
