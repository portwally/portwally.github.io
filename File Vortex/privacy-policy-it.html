File Vortex

File Vortex è stato progettato con la privacy come principio fondamentale. Tutta l'elaborazione dei file e l'analisi IA avviene interamente sul tuo dispositivo. Nessun dato viene raccolto, trasmesso o condiviso con nessuno.

Raccolta Dati

File Vortex non raccoglie, memorizza o trasmette alcun dato personale. L'applicazione funziona interamente offline e localmente sul tuo Mac.

In particolare, File Vortex non raccoglie:

  • Informazioni personali (nome, e-mail, indirizzo, ecc.)
  • Analisi di utilizzo o telemetria
  • Contenuto o metadati dei file
  • Rapporti di crash o diagnostica
  • Dati di posizione
  • Identificatori del dispositivo

Elaborazione sul Dispositivo

Tutta l'analisi dei file, incluso il riconoscimento delle immagini tramite IA, l'estrazione del testo e la generazione di parole chiave, viene eseguita interamente sul tuo Mac utilizzando i framework Vision e NaturalLanguage integrati di Apple. Nessun file, immagine o dato estratto viene mai inviato a server esterni.

Archiviazione Locale dei Dati

File Vortex memorizza i seguenti dati localmente sul tuo Mac:

  • Database dell'indice dei file: Un database SwiftData locale contenente metadati sui tuoi file organizzati (nomi dei file, tag, descrizioni, miniature). Questi dati rimangono sul tuo dispositivo.
  • Preferenze utente: Le tue impostazioni come il percorso della cartella di archiviazione e la preferenza del formato data, memorizzate nelle preferenze locali dell'applicazione.
  • Segnalibri con ambito di sicurezza: Dati dei segnalibri di macOS che permettono all'applicazione di accedere alla cartella di archiviazione scelta tra i riavvii dell'applicazione.

Tutti i dati memorizzati localmente possono essere rimossi eliminando l'applicazione.

Accesso ai File

File Vortex accede ai file solo quando li trascini e rilasci esplicitamente sulla finestra dell'applicazione, o quando cerchi nella tua libreria indicizzata. L'applicazione accede alla cartella di archiviazione scelta (NAS, unità esterna o cartella locale) per copiare e organizzare i file. Questo accesso viene concesso da te attraverso il selettore file di macOS ed è protetto dal sandboxing di macOS.

Accesso alla Rete

File Vortex richiede l'accesso alla rete esclusivamente per comunicare con il tuo storage di rete locale (NAS) tramite il protocollo SMB. L'applicazione non si connette a server esterni, API o servizi cloud.

Servizi di Terze Parti

File Vortex non utilizza alcun servizio di terze parti, SDK, strumenti di analisi o framework pubblicitari.

Privacy dei Minori

File Vortex non raccoglie dati da nessun utente, inclusi i minori. L'applicazione è sicura per l'uso da parte di persone di tutte le età.

Modifiche a questa Informativa

Se questa informativa sulla privacy viene aggiornata, le modifiche saranno pubblicate su questa pagina con una data di decorrenza aggiornata. Poiché File Vortex non raccoglie alcun dato, cambiamenti significativi sono improbabili.
