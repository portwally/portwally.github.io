Política de Privacidad

File Vortex

← Volver a File Vortex

File Vortex fue diseñado con la privacidad como principio fundamental. Todo el procesamiento de archivos y análisis de IA ocurre completamente en tu dispositivo. No se recopilan, transmiten ni comparten datos con nadie.

Recopilación de Datos

File Vortex no recopila, almacena ni transmite ningún dato personal. La aplicación funciona completamente sin conexión y localmente en tu Mac.

Específicamente, File Vortex no recopila:

  • Información personal (nombre, correo electrónico, dirección, etc.)
  • Análisis de uso o telemetría
  • Contenido o metadatos de archivos
  • Informes de fallos o diagnósticos
  • Datos de ubicación
  • Identificadores de dispositivo

Procesamiento en el Dispositivo

Todo el análisis de archivos, incluyendo el reconocimiento de imágenes por IA, la extracción de texto y la generación de palabras clave, se realiza completamente en tu Mac utilizando los frameworks Vision y NaturalLanguage integrados de Apple. Ningún archivo, imagen o dato extraído se envía jamás a servidores externos.

Almacenamiento Local de Datos

File Vortex almacena los siguientes datos localmente en tu Mac:

  • Base de datos de índice de archivos: Una base de datos SwiftData local que contiene metadatos sobre tus archivos organizados (nombres de archivos, etiquetas, descripciones, miniaturas). Estos datos permanecen en tu dispositivo.
  • Preferencias del usuario: Tu configuración, como la ruta de la carpeta de almacenamiento y la preferencia de formato de fecha, almacenadas en las preferencias locales de la aplicación.
  • Marcadores con ámbito de seguridad: Datos de marcadores de macOS que permiten a la aplicación acceder a la carpeta de almacenamiento elegida entre reinicios de la aplicación.

Todos los datos almacenados localmente se pueden eliminar borrando la aplicación.

Acceso a Archivos

File Vortex accede a los archivos solo cuando los arrastras y sueltas explícitamente en la ventana de la aplicación, o cuando buscas en tu biblioteca indexada. La aplicación accede a la carpeta de almacenamiento elegida (NAS, unidad externa o carpeta local) para copiar y organizar archivos. Este acceso es otorgado por ti a través del selector de archivos de macOS y está protegido por el sandboxing de macOS.

Acceso a la Red

File Vortex requiere acceso a la red únicamente para comunicarse con tu almacenamiento de red local (NAS) a través del protocolo SMB. La aplicación no se conecta a ningún servidor externo, API o servicio en la nube.

Servicios de Terceros

File Vortex no utiliza ningún servicio de terceros, SDK, herramientas de análisis o frameworks de publicidad.

Privacidad de los Niños

File Vortex no recopila datos de ningún usuario, incluidos los niños. La aplicación es segura para su uso por personas de todas las edades.

Cambios en esta Política

Si esta política de privacidad se actualiza, los cambios se publicarán en esta página con una fecha de vigencia actualizada. Dado que File Vortex no recopila ningún dato, es poco probable que haya cambios significativos.
