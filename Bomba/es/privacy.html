Politica de Privacidad

Ultima actualizacion: Marzo 2026

Introduccion

Bomba ("nosotros", "nuestro" o "app") esta comprometida con la proteccion de tu privacidad. Esta politica explica como manejamos tus datos cuando utilizas nuestra aplicacion.

Datos que Recopilamos

Datos de Ubicacion

Bomba utiliza la ubicacion de tu dispositivo exclusivamente para mostrarte las gasolineras cercanas. Tu ubicacion:

  • Se procesa unicamente en tu dispositivo
  • Nunca se envia a nuestros servidores
  • Nunca se comparte con terceros
  • No se almacena ni registra

Datos de Precios

Los datos de precios de combustible se obtienen de las APIs publicas de DGEG (Portugal) y del Ministerio para la Transicion Ecologica (Espana). Estos datos son publicos y no contienen informacion personal.

Datos que NO Recopilamos

Bomba no recopila, almacena ni transmite:

  • Informacion de identificacion personal
  • Historial de ubicaciones
  • Datos de uso
  • Identificadores de dispositivo
  • Cookies o tecnologias de seguimiento

Servicios de Terceros

DGEG y Ministerio Espanol

Utilizamos las APIs de DGEG y del Ministerio para la Transicion Ecologica para obtener los precios de combustible. No se comparte ningun dato personal con estos servicios.

Apple Maps / Google Maps

Cuando eliges navegar a una estacion, la app abre Apple Maps o Google Maps. El uso de estos servicios esta sujeto a sus respectivas politicas de privacidad.

Almacenamiento de Datos

Todas las preferencias y configuraciones se almacenan localmente en tu dispositivo utilizando el almacenamiento seguro del sistema. No se envia ningun dato a servidores externos.

Seguridad

Dado que no recopilamos ni transmitimos datos personales, no existen riesgos de seguridad asociados con el almacenamiento o transmision de tus datos.

Privacidad de Menores

Bomba no recopila datos de ningun usuario, incluidos menores de 13 anos.

Cambios en Esta Politica

Podemos actualizar esta politica periodicamente. Cualquier cambio se publicara en esta pagina con una fecha de actualizacion revisada.
