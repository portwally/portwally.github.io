Política de Privacidade

Última atualização: Abril de 2026

Introdução

O Abastece ("nós", "nosso" ou "app") está comprometido com a proteção da sua privacidade. Esta política explica como tratamos os seus dados quando você usa o nosso aplicativo para consultar preços de combustível no Brasil e no Chile.

Dados que Coletamos

Dados de Localização

O Abastece usa a localização do seu aparelho exclusivamente para mostrar os postos de combustível próximos. Sua localização:

  • É processada apenas no seu próprio aparelho
  • Nunca é enviada aos nossos servidores
  • Nunca é compartilhada com terceiros
  • Não é armazenada nem registrada

O uso da localização é opcional. Você pode definir uma localização manual nas configurações do app sem precisar ativar o GPS.

Dados de Preços de Combustível

Os preços são obtidos diretamente das fontes públicas oficiais: a API da Comisión Nacional de Energía (CNE) do Chile e o Levantamento Semanal de Preços da Agência Nacional do Petróleo (ANP) do Brasil. Esses dados são públicos e não contêm nenhuma informação pessoal.

Dados que NÃO Coletamos

O Abastece não coleta, armazena nem transmite:

  • Informações de identificação pessoal
  • Histórico de localizações
  • Dados de uso ou métricas de comportamento
  • Identificadores de dispositivo
  • Cookies ou tecnologias de rastreamento
  • Informações de conta (o app não exige cadastro)

Serviços de Terceiros

ANP — Agência Nacional do Petróleo

Usamos os dados públicos da ANP (Levantamento Semanal de Preços e cadastro de revendedores) para mostrar os preços brasileiros. As consultas a esses arquivos públicos são feitas sem incluir nenhuma informação identificável do usuário.

CNE — Comisión Nacional de Energía do Chile

Usamos a API oficial da CNE para obter os preços de combustível chilenos atualizados. Nenhum dado pessoal é compartilhado com este serviço. As consultas à API são feitas sem incluir nenhuma informação identificável do usuário.

Apple Maps / Google Maps

Quando você escolhe navegar até um posto, o app abre o Apple Maps ou o Google Maps. O uso desses serviços de navegação está sujeito às respectivas políticas de privacidade, independentes do Abastece.

Armazenamento Local de Dados

As preferências do usuário (país, tipo de combustível, raio de busca, localização manual) são armazenadas localmente no seu aparelho usando o armazenamento seguro do sistema. Nenhum dado de configuração é enviado para servidores externos.

Os dados de postos podem ficar em cache no aparelho para permitir consulta sem conexão. Esse cache é local e não é compartilhado.

Segurança

Como o Abastece não coleta nem transmite dados pessoais, não existem riscos associados ao armazenamento ou à transmissão dos seus dados. A comunicação com a ANP e com a API da CNE é feita por conexões cifradas (HTTPS).

Privacidade de Menores

O Abastece não coleta dados de nenhum usuário, inclusive menores de 13 anos. O app não exige cadastro nem conta, portanto não é possível coletar dados de nenhuma pessoa.

Alterações nesta Política

Podemos atualizar esta política periodicamente. Qualquer mudança será publicada nesta página com uma data de atualização revisada. Recomendamos revisar esta página de vez em quando.
