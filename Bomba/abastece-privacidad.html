Politica de Privacidad

Ultima actualizacion: Abril 2026

Introduccion

Abastece ("nosotros", "nuestro" o "app") esta comprometida con la proteccion de tu privacidad. Esta politica explica como manejamos tus datos cuando utilizas nuestra aplicacion para consultar precios de combustible en Chile, Brasil y Argentina.

Datos que Recopilamos

Datos de Ubicacion

Abastece utiliza la ubicacion de tu dispositivo exclusivamente para mostrarte las estaciones de servicio cercanas. Tu ubicacion:

  • Se procesa unicamente en tu dispositivo
  • Nunca se envia a nuestros servidores
  • Nunca se comparte con terceros
  • No se almacena ni registra

El uso de la ubicacion es opcional. Puedes fijar una ubicacion manual en los ajustes de la app sin necesidad de activar el GPS.

Datos de Precios de Combustible

Los precios se obtienen directamente de fuentes publicas oficiales: la API de la Comision Nacional de Energia (CNE) de Chile, el Levantamento Semanal de Precos de la Agencia Nacional do Petroleo (ANP) de Brasil, y el dataset "Precios en Surtidor" de la Secretaria de Energia de Argentina. Estos datos son de caracter publico y no contienen informacion personal.

Datos que NO Recopilamos

Abastece no recopila, almacena ni transmite:

  • Informacion de identificacion personal
  • Historial de ubicaciones
  • Datos de uso o analiticas
  • Identificadores de dispositivo
  • Cookies o tecnologias de seguimiento
  • Informacion de cuenta (la app no requiere registro)

Servicios de Terceros

CNE — Comision Nacional de Energia de Chile

Utilizamos la API oficial de la CNE para obtener los precios de combustible chilenos actualizados. No se comparte ningun dato personal con este servicio. Las consultas a la API se realizan sin incluir informacion identificable del usuario.

ANP — Agencia Nacional do Petroleo de Brasil

Utilizamos los datos publicos de la ANP (Levantamento Semanal de Precos y cadastro de revendedores) para mostrar los precios brasilenos. Las consultas a estos archivos publicos se realizan sin incluir informacion identificable del usuario.

Secretaria de Energia de Argentina

Utilizamos el dataset publico "Precios en Surtidor" (Resolucion 314/2016) publicado en datos.energia.gob.ar para mostrar los precios argentinos. Las consultas a este dataset se realizan sin incluir informacion identificable del usuario. Como el portal del gobierno argentino tiene actualmente un certificado HTTPS expirado fuera de nuestro control, las consultas a ese host concreto se realizan por HTTP. Solo se transmiten datos publicos; no se envia ni se recibe informacion personal.

Apple Maps / Google Maps

Cuando eliges navegar a una estacion, la app abre Apple Maps o Google Maps. El uso de estos servicios de navegacion esta sujeto a sus respectivas politicas de privacidad, independientes de Abastece.

Almacenamiento Local de Datos

Las preferencias del usuario (pais, tipo de combustible, radio de busqueda, ubicacion manual) se almacenan localmente en tu dispositivo mediante el almacenamiento seguro del sistema. Ningun dato de configuracion se envia a servidores externos.

Los datos de estaciones pueden guardarse en cache en el dispositivo para permitir su consulta sin conexion. Esta cache es local y no se comparte.

Seguridad

Dado que Abastece no recopila ni transmite datos personales, no existen riesgos asociados con el almacenamiento o transmision de tus datos. La comunicacion con la ANP y con la API de la CNE se realiza mediante conexiones cifradas (HTTPS). La comunicacion con el portal de la Secretaria de Energia de Argentina se realiza por HTTP debido a un problema de certificado en el lado del gobierno argentino; solo se transmiten datos publicos.

Privacidad de Menores

Abastece no recopila datos de ningun usuario, incluidos menores de 13 anos. La app no requiere registro ni cuenta, por lo que no es posible recopilar datos de ninguna persona.

Cambios en Esta Politica

Podemos actualizar esta politica periodicamente. Cualquier cambio se publicara en esta pagina con una fecha de actualizacion revisada. Te recomendamos revisar esta pagina ocasionalmente.
