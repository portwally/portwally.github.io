Privacy Policy

Last updated: March 10, 2026

1. Introduction

Lightroom2Photos ("we", "our", or "the app") is committed to protecting your privacy. This Privacy Policy explains how we handle your information when you use our macOS application.

2. Core Principle

Lightroom2Photos is a fully offline application. All data processing happens locally on your Mac. No data is transmitted to external servers, no accounts are required, and no analytics or tracking is used.

3. Information We Access

3.1 Lightroom Catalog

The app reads your Adobe Lightroom Classic catalog file (.lrcat) to extract photo metadata, collection structures, and keyword hierarchies. The catalog file is opened in read-only mode and is never modified.

3.2 Photo Files

The app accesses your original photo files to:

  • Import them into Apple Photos
  • Optionally write metadata (keywords, ratings) to file copies via ExifTool

When metadata writing is enabled, the app creates temporary copies of your files, writes metadata to the copies, and imports the copies. Your original files are never modified.

3.3 Apple Photos Library

The app uses Apple's PhotoKit framework to import photos, create albums, and organize your Photos library. This requires the "Photos Library" permission which you grant explicitly.

3.4 Migration Manifests

After each migration, the app saves a manifest file locally in your Application Support directory. This file contains only asset identifiers (not photos) and is used to enable the undo/rollback feature. Manifests are stored at:

~/Library/Application Support/Lightroom2Photos/Manifests/

4. Information We Do NOT Collect

Lightroom2Photos does not collect, store, or transmit:

  • Personal identification information
  • Account or login credentials
  • Usage analytics or tracking data
  • Crash reports or diagnostics
  • Any data to external servers or cloud services
  • Advertising identifiers

5. Third-Party Services

Lightroom2Photos does not integrate with any third-party services, advertising networks, or analytics platforms. The app operates entirely offline.

The optional ExifTool dependency is a local command-line tool that runs on your machine. It does not connect to the internet.

6. Data Storage

All data remains on your device:

  • Migration manifests are stored in Application Support
  • Temporary file copies (for metadata writing) are created in a temporary directory and cleaned up after migration
  • No data is stored on external servers

7. Permissions

The app requests the following system permissions:

  • Photos Library (Read & Write): Required to import photos and create albums in Apple Photos
  • File Access: Required to read your Lightroom catalog and source photo files

You can revoke these permissions at any time in System Settings.

8. Children's Privacy

Lightroom2Photos does not collect any personal information from anyone, including children under 13. The app does not require any personal information to function.

9. Your Rights

You can:

  • Revoke Photos library access at any time in System Settings
  • Delete migration manifests from Application Support
  • Delete the app to remove all locally stored data
  • Use the built-in rollback feature to undo any migration

10. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated revision date.
