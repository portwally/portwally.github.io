Fontes de Dados

Os dados de incêndios são fornecidos por:

  • NASA FIRMS (Fire Information for Resource Management System)
  • EFFIS (Sistema Europeu de Informação sobre Incêndios Florestais) / Serviço de Gestão de Emergências Copernicus

Esta aplicação é apenas para fins informativos. Os dados de incêndios podem estar atrasados e não devem ser usados como única fonte para decisões de emergência.
