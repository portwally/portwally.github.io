Fuentes de Datos

Los datos de incendios son proporcionados por:

  • NASA FIRMS (Sistema de Información de Incendios para la Gestión de Recursos)
  • EFFIS (Sistema Europeo de Información sobre Incendios Forestales) / Servicio de Gestión de Emergencias Copernicus

Esta aplicación es solo para fines informativos. Los datos de incendios pueden estar retrasados y no deben usarse como única fuente para decisiones de emergencia.
