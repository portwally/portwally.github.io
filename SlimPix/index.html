SlimPix

Shrink Images, Not Quality

Powerful image compression for macOS. Reduce file sizes by up to 80% while maintaining visual quality. Perfect for web developers, photographers, and anyone who needs smaller images.

Features

📉

Massive Size Reduction

Reduce image file sizes by up to 80% using advanced compression algorithms while preserving visual quality.

🔒

Strip Metadata

Remove EXIF data, GPS coordinates, camera info, and other metadata to protect your privacy and reduce file size.

⚡

Batch Processing

Drag and drop multiple images at once. Process entire folders of images in seconds with progress tracking.

🎛️

Quality Control

Fine-tune compression quality with an intuitive slider. Preview results before saving to find the perfect balance.

🔄

Format Conversion

Convert between JPEG, PNG, HEIC, and WebP formats. Choose the best format for your needs.

🛡️

Safe Processing

Create optimized copies by default, keeping your originals safe. Optional in-place replacement for power users.

Advanced Compression Engines

SlimPix uses industry-leading open-source compression libraries

🚀 MozJPEG

Mozilla's optimized JPEG encoder produces files 5-10% smaller than standard encoders with no quality loss. Uses advanced Huffman table optimization and progressive encoding.

✨ Zopfli

Google's Zopfli algorithm creates the smallest possible PNG files. Takes longer but achieves 3-8% better compression than standard deflate - completely lossless.

🍎 Native Encoders

Fall back to Apple's built-in encoders for maximum speed when you need quick results. Full support for HEIC and WebP formats.
