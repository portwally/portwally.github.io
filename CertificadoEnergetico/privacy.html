Política de Privacidade

Última atualização: Março de 2026

1. Introdução

Certificado Energético ("nós", "nosso" ou "a aplicação") está empenhado em proteger a sua privacidade. Esta Política de Privacidade explica como recolhemos, utilizamos e protegemos as suas informações quando utiliza a nossa aplicação móvel.

2. Informações que Recolhemos

2.1 Dados do Edifício

A aplicação recolhe os dados que introduz sobre a sua habitação para calcular a classe energética:

  • Localização (concelho)
  • Características do edifício (área, ano de construção, tipologia)
  • Dados da envolvente térmica
  • Sistemas de climatização e AQS
  • Sistemas de energia renovável

Todos estes dados são processados localmente no seu dispositivo e não são transmitidos para servidores externos.

2.2 Definições da Aplicação

As suas preferências são armazenadas localmente no seu dispositivo utilizando os mecanismos de armazenamento padrão da Apple.

3. Dados que NÃO Recolhemos

Certificado Energético não recolhe, armazena ou transmite:

  • Informações de identificação pessoal
  • Informações de conta ou login
  • Dados de localização GPS
  • Análises de utilização ou dados de rastreamento
  • Quaisquer dados para servidores externos

4. Armazenamento de Dados

Todos os dados são armazenados localmente no seu dispositivo:

  • Os dados introduzidos são guardados em memória durante a sessão
  • Nenhum dado é armazenado em servidores externos
  • Eliminar a aplicação remove todos os dados locais

5. Privacidade das Crianças

Certificado Energético não recolhe intencionalmente qualquer informação pessoal de crianças menores de 13 anos. A aplicação não requer qualquer informação pessoal para funcionar.

6. Os Seus Direitos

Pode:

  • Eliminar a aplicação para remover todos os dados armazenados localmente
  • Utilizar a aplicação sem fornecer qualquer informação pessoal

7. Alterações a Esta Política

Podemos atualizar esta Política de Privacidade periodicamente. As alterações serão publicadas nesta página com uma data de revisão atualizada.
